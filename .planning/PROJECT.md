# EnergyBot

## What This Is

Telegram-бот с Mini App для управления личной энергией. Помогает отслеживать 4 типа энергии (физическая, ментальная, эмоциональная, духовная), понимать причины просадок и получать персонализированные рекомендации по восстановлению. Для команды/близкого круга.

## Core Value

Человек понимает какая энергия просела, почему, и получает конкретный способ её восстановить — нельзя лечить эмоциональное выгорание спортом, каждый тип энергии требует своего восстановления.

## Context

### Методология 4 энергий

**Физическая** — тело, гормоны, сон, глюкоза. Тратится через недосып, стресс, алкоголь, сидение. Восстанавливается биологией: сон 7-9ч, ходьба, силовые, белок, вода, баня/контраст.

**Ментальная** — концентрация, префронтальная кора. Тратится через многозадачность, уведомления, скроллинг, 3-5ч непрерывного фокуса. Восстанавливается расфокусом: огонь/вода/горизонт, медитация, прогулка без телефона, дневной сон 15-25 мин.

**Эмоциональная** — социальная батарея, кортизол/окситоцин/дофамин. Тратится через конфликты, токсичное окружение, критику, изоляцию. Восстанавливается через связь: близкие люди, юмор, сообщество по интересам, природа, экстрим, выплеск (крик).

**Духовная** — смысл, миссия. Может конвертироваться в любую другую. Тратится через бессмысленную работу, жизнь ради чужого мнения, цинизм. Восстанавливается через: миссию, благотворительность, помощь слабым, размышления о вечном, религия/философия.

**Ключевой принцип:** Каждый тип нагрузки требует своего восстановления. Нельзя путать способы. "Отдых — часть работы, работа — часть отдыха" (Маргулан).

### Техническая среда

- Telegram Bot API + Telegram Mini App
- Бот для круга ~20-50 человек
- Нужна база данных для хранения истории
- AI (Claude/GPT API) для персонализированных советов

## Constraints

- **Платформа**: Telegram Bot + Mini App — без отдельного приложения
- **Аудитория**: Закрытый круг, не публичный бот
- **Методология**: Строго 4 типа энергии, проверенные практики восстановления
- **AI**: Советы должны быть на основе базы знаний (методология), а не общие

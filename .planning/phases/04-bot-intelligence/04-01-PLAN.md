    1. **Types:**
    ```typescript
    interface EnergyDrop {
      energyType: EnergyType;
      currentValue: number;
      previousAvg: number;
      delta: number;       // negative number
    }

    interface DiagnosticResult {
      hasEnoughData: boolean;
      drops: EnergyDrop[];           // energy types that dropped 2+ points
      chronicLows: EnergyType[];     // types averaging below 5 over last 7 days
      lowestEnergy: { type: EnergyType; value: number } | null;  // weakest current type
      latestLog: { physical: number; mental: number; emotional: number; spiritual: number } | null;
    }
    ```

    2. **`analyzeEnergyHistory(userId: number): Promise<DiagnosticResult>`**
    - Query last 14 days of EnergyLog for the user, ordered by createdAt desc
    - If fewer than 2 logs, return `{ hasEnoughData: false, drops: [], chronicLows: [], lowestEnergy: null, latestLog: null }`
    - Latest log = most recent entry
    - Previous average = mean of entries 2-6 (up to 5 prior logs) for each energy type
    - For each energy type: if latest - previousAvg <= -2, add to drops array
    - Chronic lows: compute 7-day rolling average per type, flag if below 5
    - lowestEnergy: the type with the lowest value in the latest log

    3. **`formatDiagnostic(result: DiagnosticResult): string`**
    - If not enough data: "Пока мало данных для анализа. Продолжай отмечать энергию!"
    - Otherwise: format a readable Russian-language summary listing drops, chronic lows, and current levels
    - Use emoji sparingly (only traffic-light style: red/yellow/green circles for levels)

    1. Create `src/handlers/report.ts`:

    ```typescript
    import type { Context } from "grammy";
    import prisma from "../db.js";
    import { analyzeEnergyHistory, formatDiagnostic } from "../services/diagnostics.js";

    export async function reportHandler(ctx: Context): Promise<void> {
      const from = ctx.from;
      if (!from) return;

      const user = await prisma.user.findUnique({
        where: { telegramId: BigInt(from.id) },
      });

      if (!user) {
        await ctx.reply("Сначала отправь /start чтобы зарегистрироваться.");
        return;
      }

      const result = await analyzeEnergyHistory(user.id);
      const message = formatDiagnostic(result);
      await ctx.reply(message, { parse_mode: "HTML" });
    }
    ```

    2. In `src/bot.ts`:
    - Import `reportHandler` from `./handlers/report.js`
    - Add `bot.command("report", reportHandler);`
